# 🌍 Covid-19 Analytics Dashboard

A professional Covid-19 Data Analysis Dashboard built using Python and Streamlit.  
This project analyzes Covid-19 data and visualizes worldwide and country-wise statistics through interactive charts and dashboards.



# 🚀 Live Website

🔗 https://hexsoftwarescovid19project.streamlit.app



# 📌 Features

✅ Worldwide Covid-19 Statistics  
✅ Country-wise Covid Analysis  
✅ Interactive Dashboard  
✅ Confirmed Cases Trend Graph  
✅ Pie Chart Visualization  
✅ Top 10 Countries Analysis  
✅ Dataset Preview Section  
✅ Clean Dark Theme UI  
✅ Live Website Deployment



# 📊 Technologies Used

- Python
- Pandas
- Matplotlib
- Streamlit
